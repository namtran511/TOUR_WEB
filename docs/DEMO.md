# Demo Script - Travel Spot Finder API

## 1. Khởi động server
```bash
npm run dev
```

## 2. Mở tài liệu API
- http://localhost:5000/api-docs

## 3. Demo luồng chính
1. Đăng nhập admin
   - `admin@travelspot.com` / `admin123`
2. Lấy danh sách categories
3. Tạo một spot mới
4. Tìm kiếm spot theo keyword
5. Tìm nearby spot bằng lat/lng
6. Đăng ký user mới hoặc dùng user test
7. User review một spot
8. User add favorite
9. Xem lại spot detail để thấy rating/favorite thay đổi
10. Gọi endpoint directions của Mapbox
